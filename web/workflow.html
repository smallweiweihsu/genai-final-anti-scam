System Workflow

系統流程

本專案將可疑信件分析拆成使用者輸入、模型判斷、白話解釋與安全行動，讓模型結果能被一般使用者理解。

1

使用者輸入

使用者貼上 Email，或上傳可疑信件截圖。

2

前端接收

前端整理文字 / 圖片輸入，並保留展示狀態。

3

送出模型 API

文字內容送往模型 API；圖片未來可先經 OCR 或多模態模型處理。

4

模型回傳

模型回傳風險等級、風險分數、判斷信心與可疑特徵。

5

介面轉譯

前端將模型輸出轉成 tags、checklist 與白話解釋。

6

安全行動

使用者依照建議查證，不直接點擊可疑連結或開啟附件。

目前 prototype 狀態

目前使用 rule-based analyzer 模擬模型結果，適合展示介面流程與互動方式。

圖片功能狀態

圖片功能目前為 mock 展示；未來可串接 OCR / 多模態模型讀取截圖文字與畫面特徵。
