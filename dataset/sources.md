# 資料來源紀錄

此檔案用來記錄資料來源、授權狀態、是否可重新散布，以及是否適合放入公開 demo。

## 自製 demo 案例

- 來源：組員自製。
- 授權：專案內可使用。
- 可否公開展示：可以。
- 注意事項：使用泛稱，不使用真實品牌、真實個資或真實 logo。

## 公開資料集參考

### luongnv89/phishing-email

- 來源：Hugging Face dataset。
- 用途：B 的 phishing email detection 模型方向參考。
- 授權狀態：使用前需確認 dataset page 的授權條款。
- 專案策略：若授權明確可用，再整理進資料表；若授權不明，只記錄來源，不重新散布原始內容。

## 待補來源

後續每次新增外部資料時，請補上：

- source_name
- source_url
- access_date
- license
- allowed_use
- redistribution_allowed
- notes
